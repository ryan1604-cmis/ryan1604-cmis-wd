How to Create a Webpage on the Production Server

Here is a step-by-step list to help guide you in making a new webpage on your production server. I hope this helps you!

  1. Make a .html file using basic text editor like gedit.
  2. Validate the .html file at this link. Make sure to validate while you are making the file, not just at the end.

After you are done with your .html file and it validates with no errors.

  1. Open Terminal (or a similar program).
  2. Type: sftp ryan1604@cmis-hsteacher
    from the directory of choice.
  3. Enter your username and password. Now Terminal should display a promt like this: sftp>
  4. Type: cd Public/html
    to move to the folder that you want to be in, and then type:
    put *yourfilenamehere*.html
  5. Then, to check that your .html file has been put on the production server type:
    http://cmis-hsteacher/~*name1234*/*foldername*
    If the desired webpage is displayed on the screen, then you have suceeded. Yay!

Lists and Links Assignment Description

This assignment is to teach us how to make lists and links in a webpage using HTML 5. We had to attach serveral related links to HTML help pages as well as make both ordered and unordered list with differing contents.

  • Create a new folder called "listsandlinks" in your repository folder.
  • Inside listsandlinks create a file called "index.html" and a file called "main.css"
  • In index.html create an ordered list of step by step instructions to create a new webpage on the production server.
  • Also create an unordered list of 5 or more links to websites with useful webdesign related information.
  • In the nav section, create an unordered list of links with a link back to your main index page.
  • In your main index.html file in public_html, add an unordered list of links to the nav section with a link to "listsandlinks".
  • Add an article with a heading and a paragraph description of this assignment. Also add an unordered list that contains this list of requirements.
  • Turn in your work.
