How to Create a Webpage on the Production Server

Here is a step-by-step list to help guide you in making a new webpage on your production server. I hope this helps you!

  1. Make a .html file using basic text editor like gedit.
  2. Validate the .html file at this link. Make sure to validate while you are making the file, not just at the end.

    3. After you are done with your .html file and it validates with no errors.

  1. Open Terminal (or a similar program).
  2. Type: sftp ryan1604@cmis-hsteacher
    from the directory of choice.
  3. Enter your username and password. Now Terminal should display a promt like this: sftp>
  4. Type: cd Public/html
    to move to the folder that you want to be in, and then type:
    put *yourfilenamehere*.html
